# 课程实战项目索引

所有实操均按课程顺序独立成目录。模型与数据不会由脚本下载，每个目录的 `README.md` 都列出了需要提前准备的本地模型和数据。

| 序号 | 目录 | 内容 |
| --- | --- | --- |
| 1 | `practice_01_env_basic_call` | 环境说明、GPU 查看、基础 VLM 推理与结果保存 |
| 2 | `practice_02_sam_segmentation` | SAM/SAM3 点、框、文本提示分割与可视化导出 |
| 3 | `practice_03_qwen_image_vl` | Qwen-Image 生成/编辑与 Qwen-VL / InternVL VQA、OCR、JSON 输出 |
| 4 | `practice_04_dataset_prelabel_convert` | 数据目录构造、预标注、COCO/YOLO/JSONL 转换 |
| 5 | `practice_05_train_lora` | 基础视觉训练与 VLM LoRA 微调 |
| 6 | `practice_06_component_tuning` | Projector-only、Projector + LLM、Vision Tower 局部解冻参数对比 |
| 7 | `practice_07_sam_finetune_compare` | SAM/SAM3 分割模块微调、前后效果对比、COCO 导出 |
| 8 | `practice_08_vpt_multi_gpu` | Visual Prompt Tuning 与 torchrun / Accelerate 多 GPU 启动 |
| 9 | `practice_09_vqa_app_flow` | 多轮多模态 VQA 应用流程与结构化输出 |
| 10 | `practice_10_model_service_deploy` | FastAPI 服务封装、基准测试、模型合并与端到端流程 |
