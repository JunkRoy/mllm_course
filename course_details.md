### 课程实战-1：视觉大模型环境说明与基础调用实操（1h）

	• 介绍预置实验环境、模型目录、数据目录、推理脚本目录和运行方式。 
	• 演示 GPU 状态查看、模型加载、输入图片路径配置、输出结果保存方式。 
	• 实操运行基础测试脚本，完成环境连通性验证和模型推理结果检查。 
	• 实操修改输入图片和基础参数，观察模型输出结果变化。 

### 课程实战-2：SAM/SAM3 目标分割与可视化实操（1h） 

	• 演示使用 SAM/SAM3 完成点提示、框提示、文本提示下的目标分割。 
	• 演示查看模型输出的 mask、bbox、分割区域和可视化图片。 
	• 实操对指定图片完成目标分割，导出 mask、bbox 和可视化结果。 
	• 实操对比分割结果，观察小目标、遮挡、边界不清晰等场景下的分割效果差异。 

### 课程实战-3：Qwen-Image、Qwen-VL / InternVL 模型调用实操（1h） 

	• 演示使用 Qwen-Image 完成文本生成图像、图像编辑和指定元素生成。 
	• 实操根据给定提示词生成样本图片，尝试控制场景、对象、文字和风格。 
	• 演示使用 Qwen-VL / InternVL 完成图像描述、目标识别、OCR 识别和结构化输出。 
	• 实操输入图片和文本问题，完成图片问答、文字识别和 JSON 格式输出。 
 
### 课程实战-4：视觉任务数据构造、自动预标注与格式转换实操（1h） 

	• 演示图片目录、标注文件、训练集、验证集、测试集的组织方式。 
	• 实操完成分类、检测、分割、OCR 或 VQA 数据目录构建。 
	• 演示使用 SAM/SAM3、Grounding DINO 对图片进行目标区域预标注。 
	• 实操完成 bbox、mask、OCR 区域或 VQA 样本的预标注与修正。 
	• 实操将原始标注数据转换为 COCO、YOLO、JSONL 等训练格式。 

### 课程实战-5：基础视觉模型训练与视觉语言模型 LoRA 微调实操（1.5h） 

	• 演示运行分类、检测、分割或 OCR 的基础训练脚本。 
	• 实操启动基础训练任务，完成训练参数配置、结果保存和预测结果查看。 
	• 演示使用 image + instruction + answer 数据启动 Qwen-VL / LLaVA / InternVL 类模型 LoRA 微调。 
	• 实操配置 LoRA 参数，运行小规模微调任务，保存 adapter 权重并进行简单验证。 
	• 实操对比基础视觉模型训练与视觉语言模型微调的数据格式和训练结果差异。 

### 课程实战-6：组件选择式微调与训练参数对比实操（1h） 

	• 演示 Projector-only、Projector + LLM、Vision Tower 局部解冻等配置方式。 
	• 实操运行小规模 Projector-only 或 Projector + LLM 微调任务。 
	• 实操对比组件选择式微调与 LoRA 微调在训练参数量、显存占用、训练速度和输出效果上的差异。 
	• 实操查看训练日志、adapter 或 checkpoint 输出结果，并进行基础推理验证。 

### 课程实战-7：SAM/SAM3 分割模块微调与分割效果对比实操（1h） 

	• 演示使用 mask / polygon 数据进行 SAM/SAM3 分割相关模块微调或调优。 
	• 实操运行分割模块微调脚本，完成指定图片的分割结果验证。 
	• 实操对比微调前后 mask 边界、目标完整度、小目标和遮挡目标的分割效果。 
	• 实操将分割结果导出为 mask、bbox 或 COCO 格式，理解分割结果对检测数据质量的提升作用。 

### 课程实战-8：Visual Prompt Tuning 与多 GPU 训练实操（1h） 

	• 演示冻结视觉主干、训练视觉 prompt token 的配置方式。 
	• 实操运行 Visual Prompt Tuning 小规模训练任务，观察参数量、显存占用和训练速度。 
	• 演示使用 torchrun、DeepSpeed 或 Accelerate 启动多 GPU 训练任务。 
	• 实操运行多 GPU 微调任务，对比单 GPU 与多 GPU 的训练速度、显存占用、global batch size 和 checkpoint 保存方式。 

### 课程实战-9：多模态 VQA 应用流程搭建实操（1h） 

	• 演示搭建图片上传、文本问题输入、模型推理、答案生成和结构化输出流程。 
	• 实操使用 Qwen-VL / InternVL 完成图像问答、OCR 理解、目标识别和 JSON 格式输出。 
	• 实操修改 Prompt，控制输出格式、回答粒度、依据说明和结构化字段。 
	• 实操针对工业图片、文档截图或现场图片完成多轮视觉问答。 

### 课程实战-10：模型服务封装、推理加速与多 GPU 部署综合实操（1.5h） 
	• 演示使用 Python 封装模型调用函数，完成图片输入、文本输入、模型推理和结果解析。 
	• 实操使用 FastAPI / Flask 封装图片上传、文本输入、模型推理、JSON 返回和可视化结果接口。 
	• 演示手动完成对训练模型的合并、量化等方法，对比不同推理精度或量化方式下的显存占用和推理速度。 
	• 实操完成单 GPU 与多 GPU 推理调用，记录不同部署配置下的响应时间、显存占用和输出效果。 
	• 实操串联分割、OCR、VQA 和结构化输出，形成端到端视觉大模型应用流程。
