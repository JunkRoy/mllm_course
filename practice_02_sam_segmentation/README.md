# 实操 2：SAM/SAM3 目标分割与可视化

课时：1 小时

本节默认分割模型、图片和环境已经就绪。学习重点是理解“提示 -> mask -> bbox -> 可视化”的目标分割流程，并学会用不同提示方式观察分割效果差异。

## 学习目标

- 理解点提示、框提示和文本提示分别适合什么场景。
- 理解 mask、bbox 和可视化图的含义。
- 学会修改提示参数并观察分割区域变化。
- 学会查看导出的 mask、bbox 和可视化结果。

## 背景知识

目标分割比目标检测更细。检测通常给出矩形框，分割要进一步判断目标具体占据哪些像素。

常见输出包括：

- mask：黑白图，白色代表目标区域。
- bbox：目标外接矩形框。
- visual：把分割结果叠加到原图上，便于人工检查。

SAM/SAM3 的核心特点是“可提示”。我们可以通过一个点、一个框或一段文字告诉模型要分割什么。

## 文件认知

- `config.yaml`：保存图片路径、提示点、提示框、文本提示和输出目录。
- `segment_with_prompts.py`：分割与可视化导出脚本。
- `outputs/segments.json`：记录每个分割结果的提示来源和 bbox。
- `outputs/mask_*.png`：导出的 mask。
- `outputs/visual_*.jpg`：叠加 mask 和 bbox 后的可视化图。
- `todo.md`：课前准备事项。

## 实验详细步骤

1. 进入目录：

```bash
cd practice_02_sam_segmentation
```

2. 查看提示配置：

```bash
sed -n '1,160p' config.yaml
```

3. 运行分割脚本：

```bash
python segment_with_prompts.py --config config.yaml
```

4. 查看输出文件：

```bash
ls outputs
cat outputs/segments.json
```

5. 修改 `box_prompts`，分别尝试：

- 框刚好包住目标。
- 框比目标大很多。
- 框只覆盖目标的一部分。

6. 修改 `point_prompts`，分别尝试：

- 点在目标中心。
- 点在目标边界。
- 点落在背景区域。

7. 对比 `mask_*.png` 与 `visual_*.jpg`，判断哪种提示更稳定。
